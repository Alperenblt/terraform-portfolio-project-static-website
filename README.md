Terraform Portfolio Static Website

Overview

This project demonstrates how to deploy and manage a static portfolio website infrastructure using Terraform and AWS services.

The goal of the project is to practice Infrastructure as Code (IaC) principles while building a scalable and cloud-based static website architecture.

⸻

Technologies Used

* Terraform
* AWS
* Next.js
* Amazon S3
* Amazon CloudFront
* Route53
* Git & GitHub

⸻

Project Features

* Infrastructure as Code (IaC)
* Static website hosting
* Cloud-based deployment architecture
* Terraform-managed infrastructure
* Version-controlled infrastructure setup
* Portfolio website deployment
